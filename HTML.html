1주차 ~ 2주차 HTML 배우기

첫 스터디로 HTML시작 부분을 공부했습니다.
코로나19의 확산세에 저희 윤곽성 프로그래밍은 온라인 Zoom 화상회의를 통해 스터디를 진행했습니다.

HTML이란 Hypertext Markup Language의 약자로 인터넷 서비스의 하나인 월드 와이드 웹(www)을 통해 볼 수 있는 문서를 만들 때 사용하는 웹 언어의 한 종류입니다. 우리가 보는 대부분의 웹 페이지들은 이 HTML로 작성되고 있습니다. HTML에서 사용하는 명령어는 태그(tag)라고 하고 이 태그들을 이용하여 여러 기능을 구현하며 웹 사이트를 만들어 갑니다.

1주차부터 2주차까지는 HTML에 익숙해지며 기초지식을 배우는 단계로
Visual Studio Code 개발 프로그램을 사용해 html의 기본구조인 head 와 body부분을 만들고
h1, a, li, ol, br, p 등 활용 빈도수가 많은 태그들 위주로 실습을 진행했습니다.
